How It Works

  1. Connect Wallet (Celo-based or EVM-compatible)
  2. Create or Receive Payment Request
  3. Select Mento Stablecoin (e.g., cUSD, cEUR, cREAL)
  4. Send/Receive Instantly
  5. Track Payment Status: Paid, Pending, Reversed
